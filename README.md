# ABD Premium — Salla Twilight Theme (Starter)

Premium RTL-first theme skeleton for Salla Twilight.

## Quick Start (Partners Portal)
1. Create/Import theme and connect GitHub.
2. Push this repo and import it — `twilight.json` will be detected.
3. In theme settings, set colors/logo and enable components.

## Structure
- `src/views/layouts/master.twig` — base layout
- `src/views/pages/*.twig` — pages
- `src/views/components/**` — reusable components
- `src/assets/styles/*.css` — CSS
- `src/assets/js/*.js` — JavaScript
- `src/locales/*.json` — translations
- `twilight.json` — theme metadata, settings, features, components

## Notes
- Colors/spacing are controlled via CSS variables and theme settings.
- Web Components are auto-injected by Twilight.
